# 📊 Sales Analytics and Reporting Tool

A Python-based project that analyzes sales data from CSV files and generates useful business insights, reports, visualizations, and sales forecasting.

## 🚀 Features

- Load sales data from CSV
- Handle missing columns dynamically
- Clean and preprocess data
- Calculate:
  - Total Revenue
  - Total Profit
  - Category-wise Sales
  - Product-wise Sales
  - Region-wise Sales
  - Monthly Sales Trend
- Generate charts automatically
- Forecast future sales using Linear Regression
- Generate PDF report
- Export cleaned data and reports

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- NumPy
- Scikit-learn
- ReportLab

## ▶️ How to Run

### 1. Install dependencies

pip install pandas matplotlib numpy scikit-learn reportlab

### 2. Run the program
python sales_analysis.py

### 3. Enter CSV file path

Example:
sales_data.csv
📌 Output

The project automatically generates:

- 📄 CSV reports (summary, category, product, etc.)
- 📊 Charts (PNG images)
- 🔮 Sales forecast file
- 📑 PDF report
- All outputs are saved in the outputs/ folder.

## 💡 Key Concept

- This tool is designed to work even with incomplete datasets.
- It dynamically checks available columns and performs only valid analysis.
